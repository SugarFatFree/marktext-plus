# MarkText Plus - 产品需求文档 (PRD)

**版本号**: V1.1.2  
**文档日期**: 2026-04-17  
**项目类型**: Flutter 桌面应用  
**更新类型**: 功能增强 + Bug 修复

---

## 1. 版本概述

### 1.1 版本背景
V1.1.2 在 V1.1.1 基础上扩展主题系统，新增 3 个主题使浅色/深色各 4 个达到平衡，同时修复预览视图的文本选择和链接交互问题。

### 1.2 版本目标
- 新增 3 个主题：粉樱（Pink Blossom）、天蓝（Sky Blue）、午夜蓝（Midnight）
- 修复预览视图中混合 CJK/Latin 文本选中高亮异常
- 修复标签页标题字体粗细不一致
- 实现预览视图 Ctrl+Click / Cmd+Click 打开链接

### 1.3 影响范围
- 主题系统（新增 3 个主题，12 语言本地化）
- 预览渲染器（选区高亮、链接交互）
- 标签页组件

---

## 2. 核心需求

### 2.1 新增主题

| 主题名 | 类型 | 风格 |
|--------|------|------|
| Pink Blossom（粉樱） | 浅色 | 柔和粉色调，近白背景 + 玫红强调色 |
| Sky Blue（天蓝） | 浅色 | 清新蓝色调，近白背景 + 蓝色强调色 |
| Midnight（午夜蓝） | 深色 | GitHub 风格深蓝，深邃背景 + 亮蓝强调色 |

主题平衡：浅色 4 个（Red Graphite, Shibuya, Pink Blossom, Sky Blue）+ 深色 4 个（Dark Graphite, Dieci OLED, Nord, Midnight）

### 2.2 预览视图链接交互
- Ctrl+Click（Windows/Linux）或 Cmd+Click（macOS）打开链接
- HTTP/HTTPS 链接在外部浏览器打开
- 本地文件路径在新标签页打开
- 不按修饰键时，链接文本可正常选中高亮

### 2.3 Bug 修复
- 预览视图混合文本选中高亮异常
- 标签页标题字体粗细不一致
